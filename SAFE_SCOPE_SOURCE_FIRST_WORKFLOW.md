## Enhanced Source Acceptance Criteria
1. **Verification Evidence:** Candidates must include a >25 word excerpt confirming incident-specific facts.
2. **Connectivity:** Must return HTTP 200.
3. **Specificity:** Generic landing pages (OSHA accidentsearch, MSHA search, etc.) are strictly rejected unless they are direct incident-specific deep links.
4. **Content Check:** Evidence must contain specific incident terms (e.g., employer, injury type, accident).
5. **Notes Quality:** Generic notes like "verified official source" are rejected.
6. **Strict Mode:** Audit runs with stricter criteria to ensure DB ingestion quality.
