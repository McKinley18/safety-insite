# ReviewCore Knowledge Ingestion (P9)

## Purpose
Create a governed workflow for ingesting safety knowledge.

## Lifecycle
- **Draft:** Newly ingested content, awaiting review.
- **Approved:** Verified content available for retrieval.
- **Needs Review:** Content missing citations or with unclear jurisdiction.

## Rules
- All ingested content is advisory-only.
- Guardrails prevent final decision-making.
- Unapproved records are not retrieved.
